04 Runtime responsibilities

Rule engine

Check errors, warnings, completeness, missing allocations, unverified controls, orphan requirements, and missing acceptance criteria.

Impact analysis

Use dependencies, behavior links, scenario coverage, verification links, DSM, and DHF state to propose a defensible re-review scope.

Interchange
