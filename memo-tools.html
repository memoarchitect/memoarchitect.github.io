Runtime responsibilities

What the tools compute

Semantic model build

Load ontology and methodology packages, parse project SysML, preserve stable identifiers, resolve relationships, and build a queryable evidence graph.

Closure and gap checks

Find missing allocations, unverified controls, orphan requirements, ambiguous interfaces, missing acceptance criteria, and incomplete scenario coverage.

Delta evidence analysis

Use structural dependencies, behavior links, scenario coverage, and verification links to propose a defensible re-review and re-verification scope.

Interchange and automation

Support import/export, JSON model output, deterministic artifact generation, CI validation, and future connectors for ALM, QMS, and test systems.
